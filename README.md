## Privacidade  

Privacidade de dados é diferente de segurança de dados. Técnicas de privacidade de dados permitem aprendizado e uso dos dados sem revelar informações sensíveis. Enquanto Segurança de dados evita acesso e exposição, então não podemos usá-los. 

Existem muitas técnicas de privacidade de dados, como de-identificação (anonimização) e agregação de dados. Essas são chamadas de métodos ad hoc de privacidade, pois não há uma regra geral, depende do dados. Ou seja, são soluções caso a caso. 

de-identificação é o mesmo que anonimização. Se baseia em avaliar o conjunto de dados e retirar informações que identifiquem indivíduos diretamente. Na prática, todas as informações podem ser usadas para identificar os indivíduos, mas os que identificam diretamente são chamados de PII (Personally Identifier Information) ou Informação de Identificação Pessoal.
