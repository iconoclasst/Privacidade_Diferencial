## De-identificação  

Significa anonimização ou pseudoanonimização. Se baseia em avaliar o conjunto de dados e retirar informações de identificação direta de indivíduos (PII).

### Exemplo  
Dado um dataset com as colunas nome, idade e CPF, fazer uma de-identificação seria retirar desse conjunto de dados as colunas nome e CPF.
