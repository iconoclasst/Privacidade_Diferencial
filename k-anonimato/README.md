## K-Anonimato

Em um conjunto de dados, selecionamos um subconjunto de quase-identificadores (qi), que podem ser usados em combinações para achar informações de indivíduos, e definimos um valor k. Esse conjunto de dados é k-anônimo se, para cada registro formado pelas colunas de quase-identificadores, todos eles aparecem pelo menos k-vezes. Para um subconjunto QI = {qi1, qi2, ..., qin}, cada registro é formado por uma sequência de tamanho n dos quase-identificadores (v(qi1), v(qi2), v(qi3), ..., v(qin)) e todos os registros aparecem pelo menos k vezes.
